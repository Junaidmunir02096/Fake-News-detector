# Fake News Detection System

This project is based on Machine Learning for fake news detection.

## Features
- Fake News Classification
- Text Preprocessing
- TF-IDF Vectorization
- Model Training
- Accuracy Comparison
- Custom News Prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Models Used
- Logistic Regression
- Naive Bayes
- Linear SVM
- Random Forest

## Dataset
Fake and Real News Dataset from Kaggle
